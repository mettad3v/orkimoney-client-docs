---
title: 'Query Parameters'
description: 'Supported onramp and offramp parameters.'
icon: 'ampersand'
---

<Warning>
`apiKey` is required for both onramp and offramp
</Warning>


| Property       | Product Type  | Description                                                                             | Type   |   |
|----------------|---------------|-----------------------------------------------------------------------------------------|--------|---|
| apiKey         | BUY,SELL      | The Partner API Key                                                                     | string |   |
| mode           | BUY,SELL,SWAP | The product modes to enable on the widget                                               | string |   |
| fiat           | BUY,SELL      | The fiatcurrency e.g EUR,USD,INR                                                        | string |   |
| cryptocurrency | BUY,SELL      | The cryptocurrency e.g BTC,ETH                                                          | string |   |
| network        | BUY,SELL      | The cryptocurrency network e.g bitcoin, ethereum                                        | string |   |
| paymentMethod  | BUY,SELL      | The payment method e.g credit_debit_card, IMPS                                          | string |   |
| fiatAmount     | BUY,SELL      |                                                                                         | number |   |
| cryptoAmount   | BUY,SELL      |                                                                                         | number |   |
| walletAddress  | BUY           |                                                                                         | string |   |
| redirectUrl    | BUY,SELL,SWAP |                                                                                         | string |   |
| sessionId      | BUY,SELL      | Used in tracking the same session across multiple devices                               | string |   |
| partnerContext | BUY,SELL      | A custom ID that can be passed into the request and also will be available in webhooks  | string |   |
